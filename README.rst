Snakedoc
========

A `Sphinx`_ extension for `Snakemake`_ workflows.

**WIP!**

Take it for a spin
------------------

.. code-block:: shell

   git clone git@github.com:smutch/snakedoc.git
   cd snakedoc
   poetry install
   poetry shell
   cd example/docs
   make html
   open build/html/index.html


.. _`Sphinx`: https://www.sphinx-doc.org/
.. _`Snakemake`: https://snakemake.readthedocs.io/
